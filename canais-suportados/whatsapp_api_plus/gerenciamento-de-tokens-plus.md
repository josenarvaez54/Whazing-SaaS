# Gerenciamento de Tokens Plus

O **Gerenciamento de Tokens Plus** foi criado para clientes que adquiriram **tokens avulsos ou pacotes de tokens** e precisam administrá-los de forma prática dentro do sistema.

O objetivo é permitir que os tokens sejam **atribuídos automaticamente aos canais**, tanto ao **cadastrar novos canais** quanto ao **migrar entre APIs**, tornando todo o processo **transparente e simples** no dia a dia.

Assim, o cliente consegue cadastrar e gerenciar canais **sem precisar informar dados adicionais**, facilitando a configuração e o atendimento.

***

## 📍 Onde encontrar essa opção?

Essa funcionalidade está disponível no **Painel SaaS**, na aba **Canais**.

***

## ⚙️ Como funciona?

O processo é bem simples:

1. Adicione um novo token
2. Informe o **token**
3. Selecione o **servidor**

## 🔄 Uso automático dos tokens

* Ao **cadastrar um novo canal** ou **migrar um canal entre APIs**, o sistema irá:
  * Buscar automaticamente um **token disponível**
  * Atribuir esse token ao canal
* Caso **não exista nenhum token disponível**, o sistema exibirá um **erro** informando a indisponibilidade.
* Ao **apagar um canal** ou **migrar para outra API**, o token utilizado:
  * Será **liberado automaticamente**
  * Ficará disponível para uso em outro canal

***

## 🔍 Consulta e controle

É possível:

* Buscar tokens por **canal**
* Verificar quais tokens estão **em uso**
* Identificar tokens **disponíveis**
