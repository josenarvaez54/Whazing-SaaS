# Formatação de Texto

## Formatação de Texto na Tela de Atendimento

Na tela de atendimento é possível **formatar mensagens antes de enviá-las**, deixando o texto mais organizado e fácil de entender para o cliente.

Você pode aplicar formatações como:

* **Negrito**
* _Itálico_
* ~~Tachado~~
* `Código`
* Citação

Essas formatações funcionam de forma **semelhante ao WhatsApp Web**.

***

## 1. Como usar a barra de formatação

A barra de formatação aparece automaticamente quando você **seleciona um texto** dentro da caixa de mensagem.

#### Passo a passo

1. Digite sua mensagem normalmente.
2. Selecione com o mouse o texto que deseja formatar e clique no valor selecionado.
3. Ao selecionar, aparecerá uma **barra de ferramentas acima do texto**.
4. Clique na formatação desejada.

## 2. Tipos de formatação disponíveis

Veja abaixo os tipos de formatação que podem ser usados.

***

### Negrito

Destaca partes importantes da mensagem.

**Exemplo**

Mensagem digitada:

```
Olá, seu pedido foi confirmado
```

Após aplicar negrito:

```
Olá, seu *pedido foi confirmado*
```

***

### Itálico

Usado para dar ênfase em palavras ou frases.

Exemplo:

```
Seu pedido está em _separação_
```

***

### Tachado

Usado para mostrar que algo foi cancelado ou alterado.

Exemplo:

```
Valor antigo ~R$120~
Novo valor R$90
```

***

### Código

Exibe o texto em formato de código ou destaque técnico.

Muito usado para:

* números de protocolo
* códigos
* comandos

Exemplo:

```
Seu protocolo é `458921`
```

***

### Citação

Permite destacar uma mensagem como citação.

Exemplo:

```
> Seu pedido será enviado amanhã
```

***

***

## 4. Dicas de uso no atendimento

Usar formatação ajuda a tornar o atendimento mais claro.

Exemplos práticos:

**Destacar informações importantes**

```
Seu pedido foi *confirmado com sucesso*.
```

**Destacar valores**

```
Valor total: *R$ 89,90*
```

**Mostrar código ou protocolo**

```
Protocolo de atendimento: `847291`
```

**Responder citando o cliente**

```
> Gostaria de saber o prazo de entrega

Seu prazo é de 3 dias úteis.
```

***

## Conclusão

A formatação de texto permite:

✔ Destacar informações importantes\
✔ Melhorar a leitura das mensagens\
✔ Organizar melhor o atendimento\
✔ Tornar a comunicação mais profissional

***

✅ Dica: utilize **negrito para informações importantes**, como valores, prazos e confirmações.
