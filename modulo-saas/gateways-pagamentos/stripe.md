# Stripe

Para configurar o Stripe:

1. Acesse o painel da Stripe
2. Vá em **Developers > API Keys**
3. Copie a **Secret Key**
4. Cole no painel do Whazing na parte de pagamento Stripe

#### 💱 Moeda (campo "Moeda")

* Por padrão vem: **USD**
* Você pode alterar para outras moedas suportadas

📚 Documentação oficial:\
[https://stripe.com/docs/currencies](https://stripe.com/docs/currencies)

📌 Exemplos de moedas:

* USD (Dólar)
* BRL (Real)
* EUR (Euro)

✅ Após configurar, os links de cobrança já serão gerados automaticamente com a moeda definida.
